How To Use

Step 1: Create a Layout

Creating a layout is simple. First, select the 'Layouts' option from the home page.

Create a new layout by hitting the 'ADD' button.

Enter an appropriate name and description, then hit the save button.

Step 2: Adding Controls

To add a control, hit the 'ADD' button in the 'Controls' section of the layout setting page. A new section will appear underneath the main layout settings to add or update a control.

Enter values into each field as instructed. Hover over the '' to get more info on the field. Hit 'SAVE' when completed.

The control now appears as part of the layout. Add as many controls as desired!

Step 3: Generating a Report

With a created layout and control, we are ready to generate reports. In the main menu, select the 'Generate Reports from Excel' option.

Select a layout for report creation. In this example, the 'test_layout' layout is used. When the report is created, we're expecting it to pull the value contained in D5 and present this as the purchase price on the report for a client named 'Bigglesworth Inc.'. A workbook is supplied using the 'Source File' option with the data that we want to present in the report. Hit the 'CREATE REPORT' button when ready.

A report is generated using the supplied source data and layout that was selected. Hover over the '' icon to view the description that was set on the control.
